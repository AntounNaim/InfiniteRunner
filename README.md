## Extensions Implemented (All working)

- **B** – Main menu scene with Start and Quit buttons
- **C** – Pause menu (Esc toggles pause, Resume and Quit to Main Menu)
- **D** – High score saved between runs and game launches (PlayerPrefs)
- **E** – Particle effect on coin pickup
- **F** – Multiple new custom chunk prefabs (4 new chunks)

## Controls

| Action | Key |
|--------|-----|
| Switch lanes (Left) | A or Left Arrow |
| Switch lanes (Right) | D or Right Arrow |
| Jump | W or Up Arrow |
| Restart (when game over) | R |
| Pause / Resume | Esc |

## Features

- Collision detection with obstacles → Game Over freeze
- Restart with R key or on-screen button
- Distance score HUD (updates during play)
- Final distance, final coins, highest coins, and high score on Game Over panel
- New High Score HUD notification when beating your best distance
- Coin collectibles with bobbing animation, rotation, sound effect, and particle effect
- Jump animation (triggered when jumping, returns to run)
- Persistent save using PlayerPrefs (scores remain after closing the game)

## Known Bugs

None. The game runs as intended.

## How to Play

Avoid obstacles, collect coins, and run as far as possible. Your distance score increases over time. Coins add to your total. Try to beat your personal best distance and collect the most coins!
